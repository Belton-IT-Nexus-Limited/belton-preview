Essential Eight Implementation Guide

The Australian Cyber Security Centre's top strategies to mitigate cyber security incidents. A practical guide for implementing all eight controls.

What's Included

The Essential Eight is a prioritised list of mitigation strategies from the ACSC. Implementing these eight strategies makes it much harder for adversaries to compromise your systems.

  • Application Control implementation
  • Patch Applications procedures
  • Microsoft Office Macro Settings
  • User Application Hardening
  • Restrict Administrative Privileges
  • Patch Operating Systems
  • Multi-Factor Authentication
  • Regular Backups

Each control includes actionable checklists and notes sections to track your implementation progress towards your target maturity level.
